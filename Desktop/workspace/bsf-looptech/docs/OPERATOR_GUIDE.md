# BSF-LoopTech オペレーター操作ガイド

## はじめに

### このガイドについて

本ガイドは、BSF-LoopTech（廃棄物処理配合最適化システム）を日常的に使用するオペレーター向けの操作マニュアルです。
画面上のボタン操作を中心に、よく使う機能を手順付きで説明します。
ターミナル操作やシステム管理の知識は不要です。

### 画面構成の概要

システムは **5つのタブ** と **AIチャット** で構成されています。

| タブ番号 | タブ名 | 主な用途 |
|---------|--------|---------|
| 0 | 搬入予定 | 搬入物の受け入れスケジュール管理 |
| 1 | 配合管理 | AI配合推薦、レシピ管理、最適化 |
| 2 | 分析ダッシュボード | KPI確認、トレンド分析、予測結果 |
| 3 | 品質管理 | 規制基準チェック、合否判定履歴 |
| 4 | マスタ管理 | 業者・固化材・溶出抑制剤・搬入物のマスタデータ |

画面右下に **AIチャットボタン**（青い丸ボタン）が常時表示されます。
いつでもクリックしてAIアシスタントに質問できます。

### ログイン方法

1. ブラウザで システムのURL にアクセスします
2. ログイン画面が表示されます
3. **ユーザー名** と **パスワード** を入力します
4. **「ログイン」** ボタンをクリックします
5. ログイン成功後、メイン画面（搬入予定タブ）が表示されます

ログアウトするには、画面右上のユーザーアイコンをクリックし、**「ログアウト」** を選択します。

## 1. 搬入管理（Tab 0）

搬入予定タブでは、廃棄物の受け入れスケジュールを管理します。
一覧表示のほか、週間・2週間・月間のカレンダー表示にも対応しています。

### 1.1 搬入予定の確認

1. 画面上部の **「搬入予定」** タブをクリックします
2. 搬入予定の一覧が表示されます
3. 表示形式を切り替えるには、一覧上部の切替ボタンを使用します
   - **テーブル表示**: リスト形式で全件確認
   - **1週間**: 週間カレンダー表示
   - **2週間**: 2週間カレンダー表示
   - **1ヶ月**: 月間カレンダー表示
4. カレンダー表示では、前後の期間に移動するボタンと「今日」ボタンが使えます

### 1.2 搬入予定の新規登録

1. 一覧画面で **「＋ 新規登録」** ボタンをクリックします
2. 登録フォームが表示されます
3. 以下の項目を入力します
   - **搬入日**: 搬入予定日を選択
   - **搬入先**: プルダウンから選択（マスタ管理で登録済みの業者）
   - **搬入物**: プルダウンから選択（搬入先に応じて絞り込まれます）
   - **予定数量**: 搬入予定の数量を入力
   - **備考**: 必要に応じてメモを入力
4. **「保存」** ボタンをクリックします
5. 一覧に新しい搬入予定が追加されます

### 1.3 搬入予定の編集・コピー

**編集する場合:**
1. 一覧で対象行の **鉛筆アイコン** をクリックします
2. 編集フォームが開きます
3. 内容を変更して **「保存」** をクリックします

**コピーして新規作成する場合:**
1. 一覧で対象行の **コピーアイコン** をクリックします
2. コピーされた内容でフォームが開きます
3. 日付や数量を変更して **「保存」** をクリックします

### 1.4 ステータスの更新

搬入予定には以下のステータスがあります。

| ステータス | 表示 | 説明 |
|-----------|------|------|
| 予定 | グレーのバッジ | 搬入予定として登録された状態 |
| 搬入済 | 緑のバッジ | 実際に搬入が完了した状態 |
| キャンセル | 赤のバッジ | 搬入が取り消された状態 |

**搬入済みに変更する場合:**
1. 一覧で対象行の **チェックマークアイコン** をクリックします
2. 確認ダイアログが表示されます
3. 搬入日と搬入量を確認・修正します
4. **「搬入済みにする」** をクリックします
5. ステータスが「搬入済」に変わり、搬入記録が自動作成されます

**キャンセルする場合:**
1. 一覧で対象行の **×アイコン** をクリックします
2. 確認ダイアログで **「キャンセルする」** をクリックします

### 1.5 配合管理への連携

搬入済みの記録から配合管理へ進むことができます。

1. ステータスが「搬入済」の行で **「配合開始」** ボタンをクリックします
2. 自動的に **配合管理タブ** に切り替わります
3. AI配合推薦ダイアログが自動で開きます

### 1.6 検索とフィルタ

1. 一覧上部のフィルタ条件を利用します
   - ステータスでの絞り込み
   - 期間での絞り込み
2. 条件を変更すると、自動的に一覧が更新されます

### 1.7 CSV出力

1. 一覧画面で **「CSV出力」** ボタン（ダウンロードアイコン）をクリックします
2. 現在の検索条件に一致するデータがCSVファイルとしてダウンロードされます
3. ダウンロードされたファイルはExcelで開くことができます（BOM付きUTF-8形式）

## 2. 配合管理（Tab 1）

配合管理タブでは、搬入物に対する配合の決定・承認・適用を管理します。
AIによる配合推薦機能も利用できます。

### 2.1 配合一覧の確認

1. 画面上部の **「配合管理」** タブをクリックします
2. 配合記録の一覧が表示されます（上部セクション）
3. ステータスやソースで絞り込みができます
   - **ステータス**: 提案 / 承認済 / 適用済 / 検証済 / 却下
   - **ソース**: 手動 / ML予測 / 類似度 / ルール / 最適化 / レシピ

### 2.2 AI配合推薦の実行

**搬入管理タブからの起動（推奨）:**
1. 搬入管理タブで「搬入済」の記録の **「配合開始」** ボタンをクリックします
2. 自動的に配合管理タブに切り替わり、AI推薦ダイアログが開きます

**配合管理タブからの起動:**
1. 配合一覧の上部にある **「＋ AI推薦」** ボタンをクリックします
2. AI推薦ダイアログが開きます

**推薦結果の確認と選択:**
1. ダイアログに複数の推薦案が表示されます
   - **ML予測**: 過去のデータに基づくAI予測
   - **類似度**: 類似した過去の配合パターン
   - **ルール**: 規制基準に基づく推薦
2. 各推薦案の配合比率・予測コスト・信頼度を確認します
3. 採用したい推薦案の **「採用」** ボタンをクリックします
4. 配合記録が「提案」ステータスで作成されます

### 2.3 配合の承認・却下

**承認する場合:**
1. ステータスが「提案」の配合記録で **チェックマークアイコン**（承認）をクリックします
2. 確認ダイアログで **「承認する」** をクリックします
3. ステータスが「承認済」に変わります

**却下する場合:**
1. ステータスが「提案」の配合記録で **×アイコン**（却下）をクリックします
2. 確認ダイアログで却下理由を入力します
3. **「却下する」** をクリックします
4. ステータスが「却下」に変わります

### 2.4 配合の適用

承認済みの配合を実際に適用する手順です。

1. ステータスが「承認済」の配合記録で **再生アイコン**（適用）をクリックします
2. 適用ダイアログで、実際の配合量や適用日時を確認・修正します
3. **「適用する」** をクリックします
4. ステータスが「適用済」に変わります

### 2.5 溶出試験結果の登録

配合適用後、溶出試験の結果を登録して検証を完了します。

1. ステータスが「適用済」の配合記録で **検証アイコン**（盾マーク）をクリックします
2. 検証ダイアログが開きます
3. 溶出試験の各項目の測定値を入力します
4. 基準値との比較結果が自動表示されます
5. **「検証完了」** をクリックします
6. ステータスが「検証済」に変わります

### 2.6 配合詳細の確認

1. 配合一覧で対象行をクリックします
2. 画面右側に詳細ドロワーが開きます
3. 以下の情報を確認できます
   - **ステータス遷移タイムライン**: 提案→承認→適用→検証の各ステップと日時
   - **配合比率**: 各材料の配合量
   - **溶出試験結果**: 検証済みの場合、試験結果が表示されます
   - **コスト情報**: 材料コストの内訳

### 2.7 レシピ管理

配合管理タブの下部セクションでは、レシピの管理ができます。

**レシピの確認:**
1. レシピ一覧が表示されます
2. ステータス（下書き / 有効 / アーカイブ）で絞り込みできます
3. 検索バーでレシピ名を検索できます

**新規レシピの作成:**
1. **「＋ 新規作成」** ボタンをクリックします
2. レシピ名、配合内容、備考を入力します
3. **「保存」** をクリックします

**バージョン履歴の確認:**
1. レシピ一覧で対象行の **時計アイコン**（履歴）をクリックします
2. バージョン履歴ダイアログが開きます
3. 各バージョン間の差分を確認できます

### 2.8 CSV出力

1. 配合一覧の **「CSV出力」** ボタンをクリックします
2. 現在表示中のデータがCSVファイルとしてダウンロードされます

## 3. 分析ダッシュボード（Tab 2）

分析ダッシュボードでは、システム全体の稼働状況やパフォーマンスを確認できます。

### 3.1 統合概要

タブを開くと最上部に **統合概要パネル** が表示されます。
搬入・配合・品質の主要指標がひと目で確認できます。

### 3.2 KPI確認

**KPIダッシュボード** では、以下の運用指標をリアルタイムで確認できます。

- **処理量**: 廃棄物の処理量推移
- **違反率**: 規制基準を超えた割合
- **材料コスト**: 配合材料にかかるコスト推移
- **処理時間**: 平均処理時間

操作手順:
1. KPIカードに現在の数値と前回比較の矢印が表示されます
2. 期間の切り替え: プルダウンから **7日 / 14日 / 30日** を選択します
3. 各KPIの詳細グラフが下部に表示されます
4. アラートがある場合、赤や黄色の通知が表示されます

KPIデータは **30秒ごと** に自動更新されます。

### 3.3 相関分析

各指標間の相関関係をグラフで確認できます。
どの要因が品質に影響しているかの分析に役立ちます。

### 3.4 トレンド分析

1. トレンド分析セクションが表示されます
2. 時系列グラフで各指標の推移を確認できます
3. 期間を変更してトレンドを比較できます

### 3.5 予測精度

MLモデルの予測精度を確認できます。
実際の結果と予測値の比較グラフが表示されます。

## 4. 品質管理（Tab 3）

品質管理タブでは、規制基準への適合状況を確認できます。

### 4.1 規制基準チェック

1. 画面上部の **「品質管理」** タブをクリックします
2. 各溶出試験項目ごとの基準値と実績値が表示されます
3. 基準値を超えた項目は赤色でハイライトされます

### 4.2 合否判定履歴

過去の溶出試験結果の履歴を確認できます。
合格・不合格の推移がグラフで表示されます。

## 5. マスタ管理（Tab 4）

マスタ管理タブでは、システムで使用する基本データ（マスタデータ）を管理します。
4つのサブタブがあります。

### 5.1 搬入先マスタ（業者）

搬入元の業者情報を管理します。

**一覧の確認:**
1. **「マスタ管理」** タブをクリックします
2. **「搬入先」** サブタブが表示されます
3. 登録済みの業者一覧が表示されます

**新規登録:**
1. **「＋ 新規登録」** ボタンをクリックします
2. 業者名、連絡先などを入力します
3. **「保存」** をクリックします

**編集:**
1. 対象行の **鉛筆アイコン** をクリックします
2. 内容を変更して **「保存」** をクリックします

**削除:**
1. 対象行の **ゴミ箱アイコン** をクリックします
2. 確認ダイアログで **「削除」** をクリックします

### 5.2 搬入物マスタ

搬入物の種類を管理します。各搬入物は搬入先（業者）に紐づいています。

1. **「搬入物」** サブタブをクリックします
2. 搬入物の一覧が表示されます
3. 新規登録・編集・削除は搬入先マスタと同じ操作です

### 5.3 固化材マスタ

配合に使用する固化材を管理します。

1. **「固化材」** サブタブをクリックします
2. 固化材の一覧が表示されます
3. 新規登録・編集・削除は搬入先マスタと同じ操作です

### 5.4 溶出抑制剤マスタ

配合に使用する溶出抑制剤を管理します。

1. **「溶出抑制剤」** サブタブをクリックします
2. 溶出抑制剤の一覧が表示されます
3. 新規登録・編集・削除は搬入先マスタと同じ操作です

### 5.5 CSVインポート・エクスポート

各マスタデータはCSVファイルで一括入出力できます。

**CSVエクスポート（出力）:**
1. 対象のサブタブを開きます
2. **「CSV出力」** ボタン（ダウンロードアイコン）をクリックします
3. CSVファイルがダウンロードされます
4. Excelで開いて確認・編集ができます

**CSVインポート（入力）:**
1. 対象のサブタブを開きます
2. **「CSVインポート」** ボタンをクリックします
3. ファイル選択ダイアログでCSVファイルを選択します
4. インポート結果が表示されます
   - 成功件数とエラー件数が通知されます
   - エラーがある場合、行番号とエラー内容が表示されます

**CSVファイルの形式について:**
- 文字コード: UTF-8（BOM付き）
- 1行目はヘッダー行（項目名）
- エクスポートしたファイルをテンプレートとして使用することを推奨します

## 6. AIチャット

画面右下の青い丸ボタンから、AIアシスタントとチャットできます。
配合に関する質問や、過去のデータに基づく相談ができます。

### 6.1 チャットの開始

1. 画面右下の **青い丸ボタン**（ロボットアイコン）をクリックします
2. 画面右側にチャットドロワーが開きます
3. 入力欄にメッセージを入力します
4. **Enter** キーまたは送信ボタンをクリックします
5. AIからの回答が表示されます

チャットを閉じるには、ドロワー外の画面をクリックするか、閉じるボタンを使います。

### 6.2 質問のコツ

AIアシスタントに効果的に質問するポイントです。

**良い質問の例:**
- 「鉛の溶出値が基準を超えた場合、どの固化材を増やせばよいですか」
- 「先月の搬入物Aに対して、最もコストが低かった配合はどれですか」
- 「六価クロムの溶出抑制に効果的な方法を教えてください」

**具体的に質問するコツ:**
- 対象の搬入物名や業者名を含める
- 溶出項目名（鉛、六価クロム、ヒ素など）を明記する
- 期間や数量などの条件を具体的に伝える

## よくある操作の流れ

### パターン1: 新規搬入物の受け入れから配合決定まで

日常業務で最も頻繁に行う一連の操作です。

1. **搬入予定の登録**
   - 搬入予定タブ → 「＋ 新規登録」 → 搬入日・業者・搬入物・数量を入力 → 保存

2. **搬入確認（搬入日当日）**
   - 搬入予定タブ → 対象行のチェックマーク → 搬入量を確認 → 「搬入済みにする」

3. **配合開始**
   - 搬入予定タブ → 搬入済みの行の「配合開始」ボタン → 自動的に配合管理タブへ

4. **AI推薦の確認と採用**
   - AI推薦ダイアログ → 推薦案を比較 → 「採用」

5. **配合の承認**
   - 配合一覧 → 提案の行のチェックマーク → 「承認する」

6. **配合の適用**
   - 配合一覧 → 承認済の行の再生アイコン → 配合量を確認 → 「適用する」

7. **溶出試験結果の登録**
   - 配合一覧 → 適用済の行の検証アイコン → 試験結果を入力 → 「検証完了」

### パターン2: マスタデータの更新

新しい業者や材料が追加された場合の操作です。

1. **マスタ管理タブ** をクリックします
2. 該当する **サブタブ**（搬入先 / 搬入物 / 固化材 / 溶出抑制剤）を選択します
3. **「＋ 新規登録」** をクリックします
4. 必要な情報を入力して **「保存」** をクリックします
5. 一覧に新しいデータが追加されたことを確認します

### パターン3: 月次レポートの確認とデータ出力

月末の報告用にデータを確認・出力する場合の操作です。

1. **分析ダッシュボード** タブをクリックします
2. KPIダッシュボードの期間を **「30日」** に変更します
3. 各KPI指標の推移を確認します
4. 必要に応じて **品質管理** タブで規制基準の適合状況を確認します
5. **搬入予定** タブに戻り、**「CSV出力」** で搬入データを出力します
6. **配合管理** タブで **「CSV出力」** で配合データを出力します

### パターン4: CSVで一括データ登録

大量のマスタデータを一括登録する場合の操作です。

1. まず **空のテンプレートを入手** します
   - マスタ管理タブ → 対象サブタブ → 「CSV出力」でテンプレートを取得
2. **Excelでデータを準備** します
   - ダウンロードしたCSVファイルをExcelで開く
   - 既存データの下に新しいデータ行を追加
   - UTF-8（BOM付き）形式で保存
3. **インポート** します
   - マスタ管理タブ → 対象サブタブ → 「CSVインポート」 → ファイルを選択
4. **結果を確認** します
   - 成功件数とエラー件数を確認
   - エラーがある場合は内容を修正して再インポート

## 画面で困ったときのヒント

| 症状 | 対処法 |
|------|--------|
| ボタンがクリックできない | 必要な項目が未入力の可能性があります。必須項目を確認してください |
| データが表示されない | ブラウザの再読み込み（F5キー）を試してください |
| ページが真っ白になった | ブラウザの再読み込みを試してください。改善しない場合は管理者に連絡してください |
| CSVファイルが文字化けする | Excelで開く際に「UTF-8」を選択してください |
| ログイン画面に戻される | セッションの有効期限が切れています。再度ログインしてください |
| エラーメッセージが表示された | メッセージの内容をメモして管理者に連絡してください |
