# BSF-LoopTech プロジェクト概要

## プロジェクト名
**BSF-LoopTech** - ブラックソルジャーフライ養殖環境監視・最適化システム

## バージョン情報
- **システムバージョン**: 0.1.0
- **最終更新日**: 2025-10-01
- **ドキュメント作成日**: 2025-10-01

---

## 1. プロジェクトの目的

BSF（ブラックソルジャーフライ）幼虫の養殖環境を包括的に監視・管理・最適化するためのIoTシステムです。環境センサーからのリアルタイムデータ収集、高度な分析、自動制御、ERP連携を統合した総合プラットフォームを提供します。

### 主要目標
1. **リアルタイム環境監視**: 温度・湿度・ガス濃度等の24時間監視
2. **データ駆動型最適化**: センサーデータ分析による養殖条件の最適化
3. **自動制御連携**: 温湿度制御システムとの自動連携
4. **基質管理**: 複数の基質タイプの配合・追跡管理
5. **ERP統合**: 生産管理システムとのシームレス連携
6. **異常検知**: 閾値ベース＋機械学習による異常検知

## 3. 主要機能モジュール

### 3.1 センサーデータ収集
- **対象センサー**:
  - 温度センサー (MLX90614: -40°C～85°C)
  - 湿度センサー (0～100% RH)
  - ガスセンサー (H2S, NH3: 0～100 kppm)
  - 圧力センサー (900～1100 hPa)
- **収集間隔**: 5分 (可変設定)
- **通信**: MQTT over TLS (QoS 1)

### 3.2 リアルタイム監視
- WebSocket接続によるリアルタイムデータ配信
- アラート通知システム (閾値ベース)
- 3Dビジュアライゼーション
- 異常検知パネル

### 3.3 データ分析
- **統計分析**: 平均値、標準偏差、相関分析
- **トレンド分析**: 時系列パターン解析
- **異常検知**: 機械学習ベース異常検知
- **予測分析**: Prophet等を用いた予測

### 3.4 基質管理
- **基質タイプ**:
  - 下水汚泥
  - 豚の糞尿
  - 鶏糞
  - おが屑
  - カスタム基質
- **バッチ管理**: 配合比率追跡、履歴管理

### 3.5 認証・セキュリティ
- JWT認証
- ロールベースアクセス制御 (RBAC)
- レート制限
- MQTT TLS/SSL暗号化
- API認証ミドルウェア
