## 💰 補助金制度の概要

### 中小企業庁 新事業進出補助金

**正式名称**: 中小企業新事業進出補助金

**目的**: 中小企業が新たな事業分野への進出や生産性向上のための設備投資・システム導入を支援

| 項目 | 内容 |
|------|------|
| **補助率** | **1/2**（中小企業） |
| **補助上限額** | **1,500万円** |
| **補助対象** | 機械装置・システム構築費、技術導入費、専門家経費など |
| **事業期間** | 交付決定後10ヶ月以内 |
| **応募回数** | 年3～4回程度の公募 |

### 本システムへの適用

**記載すべきポイント**:

```

- 菌体リン酸肥料の配合は熟練者の勘と経験に依存
- 原料の成分変動に対する対応が属人的
- 若手への技能継承が困難
- 成分のばらつきによる品質クレームが年間XX件発生

【トレーサビリティの欠如】

- 過去の配合データが蓄積されていない
- 原料成分と製品品質の相関分析ができていない
```

```
- クレーム発生時の即座の原因追跡

【データドリブンな製造体制】
- 製造データの蓄積と分析

- 継続的な改善活動の基盤構築
```

- ✅ 交付決定**後**に発注・契約すること（前倒し発注は補助対象外）
- ✅ 補助事業期間内（10ヶ月以内）に完了すること
- ✅ 証拠書類（契約書、請求書、領収書）を適切に保管

**本システムの場合**:
```
交付決定（例：2025年6月）
  ↓
システム開発契約締結（2025年6月）
  ↓
開発期間（2025年6月～2025年10月：5ヶ月）
  ↓
本番稼働・検収（2025年11月）
  ↓
実績報告提出（2025年12月）
```

### 4. 実績報告

**提出書類**:
- 実績報告書
- 経費明細書
- 証拠書類（契約書、請求書、領収書、納品書）
- システム稼働証明（画面キャプチャ等）
- 付加価値額増加の証明

### 5. 確定検査

- 事務局による書類審査
- 必要に応じて現地調査
- 補助金額の確定

### 6. 補助金入金

- 確定検査合格後、指定口座へ入金
- 交付決定から通常6～12ヶ月後

### 7. 事業化状況報告（5年間）

- 採択後5年間、毎年事業化状況を報告
- 付加価値額の実績報告（年率平均3%以上の達成確認）
- 目標未達の場合、補助金返還の可能性

## ❓ よくある質問（FAQ）

### Q1. 交付決定前に発注してしまったら？

**A**: 補助対象外となります。必ず交付決定後に発注してください。

ただし、以下は交付決定前でも可能：
- 見積取得
- 仕様検討
- ベンダー選定

### Q2. システム開発が補助事業期間内に完了しない場合は？

**A**: 期間延長申請が可能な場合があります。ただし、事前申請が必須です。

本システムの場合：
- 開発期間: 5ヶ月（余裕を持った計画）
- 補助事業期間: 10ヶ月（十分な余裕あり）

### Q3. 補助金の入金時期は？

**A**: 実績報告後、確定検査を経て入金されます。通常6～12ヶ月後です。

**対策**: つなぎ融資の活用
- 金融機関に補助金交付決定通知を提示
- つなぎ融資の検討

### Q4. 採択率はどのくらい？

**A**: 公募回により異なりますが、通常枠で40～60%程度です。

**採択率を上げるポイント**:
- 認定支援機関との共同申請
- 革新性の明確化
- 具体的な数値目標
- 実現可能な計画

### Q5. 補助金はいつ入金される？

**A**: システム完了・実績報告後、6～12ヶ月後が目安です。

**資金繰り対策**:
- つなぎ融資の活用
- 分割払い契約
- 自己資金の準備

### Q6. 他の補助金との併用は可能？

**A**: 同一経費への重複申請は不可。ただし、異なる経費であれば併用可能な場合があります。

例:
- ものづくり補助金: システム導入費用
- 人材開発支援助成金: 研修費用（別途）

### Q7. 不採択の場合、再申請は可能？

**A**: 可能です。不採択理由を分析し、改善した計画で再挑戦できます。

**再申請のポイント**:
- 不採択理由の確認（問い合わせ可能）
- 計画の見直し・改善
- 認定支援機関との再検討

### Q8. 事業化状況報告で目標未達の場合は？

**A**: 補助金返還を求められる可能性があります。

**対策**:
- 現実的な目標設定
- 継続的なモニタリング
- 早期の改善アクション
