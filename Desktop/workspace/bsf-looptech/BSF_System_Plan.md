# BSF幼虫養殖環境監視システム 実装計画書

## 1. プロジェクト概要

### 1.1 背景と目的
BSF（ブラックソルジャーフライ）幼虫養殖環境のための包括的なIoT監視・最適化システムを構築します。このシステムは環境センサーデータの収集、リアルタイム監視、高度なデータ処理と分析、基質組成の管理、および自動制御機能を提供します。

### 1.2 システム要件概要
- **環境**: 当面はオンプレミス運用、将来的にAWSクラウド環境へ移行
- **ネットワーク**: 施設内Wi-Fi接続のセンサーデバイス
- **拡張性**: 初期15エリアから最大1000エリアまで対応
- **統合**: ERP管理システムとの連携（在庫・生産データ）
- **自動制御**: 温度・湿度の自動調整システムとの連携（必須）
- **データ出力**: CSV形式のエクスポート機能
- **基質タイプ**: 下水汚泥、豚の糞尿、鶏糞、おが屑、その他（カスタム入力、詳細条件入力対応）
- **センサーデータ収集間隔**: 5分
- **異常検知**: UI上で閾値設定可能、将来的に機械学習による自動調整

### 2.2 コンポーネント詳細

#### 2.2.1 データ収集層
- **センサーネットワーク**: Wi-Fi接続のセンサーノード
- **通信プロトコル**: MQTT 3.1.1（TLS 1.3）
- **QoS**: レベル1（少なくとも1回の配信保証）
- **トピック構造**: `bsf/{farm_id}/{device_type}/{device_id}`
- **データ収集頻度**: 5分間隔
- **センサー監視パラメータ**:
  - 温度: -10～50°C
  - 湿度: 0～100 %RH
  - 圧力: 900～1100 hPa
  - H2Sガス: 0～100 kppm
  - NH3ガス: 0～100 kppm

#### 2.2.2 基質管理システム
- **基質タイプ管理**:
  - 下水汚泥（多様な条件に対応）
  - 豚の糞尿
  - 鶏糞
  - おが屑
  - その他（カスタム入力可能）
- **属性管理**: 各基質の詳細条件入力機能
- **配合比率追跡**: リアルタイム比率モニタリング
- **履歴管理**: 基質変更履歴の記録と分析

#### 2.2.3 処理層
- **データ収集エンジン**: Telegraf
- **処理エンジン**: Python（Flask/FastAPI）
- **異常検知アルゴリズム**: 閾値ベース＋将来的に機械学習拡張
- **アラート管理**: プログラマブルなアラートルール設定
- **自動制御連携**: 温湿度システムへの制御信号送信

#### 2.2.4 ストレージ層
- **主データベース**: InfluxDB 2.7+
- **スキーマ設計**: タグベースの柔軟なデータモデル
- **リテンション設計**: 長期保存ポリシー
- **バックアップ戦略**: 自動バックアップ設定

#### 2.2.5 連携層
- **ERP連携API**: RESTful API
- **データ変換**: 双方向データ変換モジュール
- **認証**: OAuth 2.0ベースの認証
- **エクスポート機能**: CSVフォーマット生成モジュール

#### 2.2.6 可視化層
- **ダッシュボード**: Grafana
- **可視化コンポーネント**:
  - リアルタイムセンサーデータパネル
  - 基質管理ビュー
  - アラート表示・管理
  - 履歴データ分析グラフ
- **異常検知表示**: カラーコードによる状態表示
